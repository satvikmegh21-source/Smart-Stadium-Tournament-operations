# Smart Stadium & Tournament Operations Platform

An enterprise-grade SaaS dashboard and tournament management platform built using:
- **Frontend**: Next.js 15 (React 19, TypeScript, TailwindCSS, Framer Motion, Recharts)
- **Backend**: Node.js & Express (TypeScript, Socket.io, Prisma, PostgreSQL)
- **Security**: JWT & Refresh Tokens, Argon2, Helmet, CORS, Express-Rate-Limit, Zod validation
- **DevOps**: Docker, Docker Compose, GitHub Actions CI/CD

## Structure
- `/backend`: Node.js API with Express & Prisma
- `/frontend`: Next.js Web App
